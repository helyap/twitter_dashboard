Twitter scraping API

This is the home page of twitter scraping API.

You can scrap tweets in time by entering your parameters at URL_BASE/search. For example, if the url of current page is http://127.0.0.1:5000, go to http://127.0.0.1:5000/search.

If you have your own twitter data scraped from twint, you can also upload here: URL_BASE/upload. For example, if the url of current page is http://127.0.0.1:5000, go to http://127.0.0.1:5000/upload.
